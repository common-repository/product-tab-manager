This plugin allows you to manage your Woocommerce product page tabs. Tabs can be renamed, removed and re-ordered on the single product page.

== Description ==

This plugin allows you to manage your Woocommerce product page tabs (Description, Additional Information and Reviews). You can rename, re-order and show/hide these tabs for the single product page. 

= Features =

* Manage default product data tabs
* Completely hide all default tabs
* Rename and reposition (sort) default tabs
* Supports all types of products

== Pro version features ==

In the premium version you can create your own custom tabs, re-order them and set their show/hide them. You can create custom tabs for all products globally or custom tab for selected product or product categories.

* Create your own custom tabs
* Use wordpress native Block editor or WYSIWYG editor to add content for custom tabs
* Create unlimited custom tabs for 
    * all products
    * all categories
    * selected products
    * selected categories
    * just one product or category
* Supports all types of products 
* Very light weight
* No CSS or JS included. Tabs will integrated in your theme seamlessly.

== Installation ==

Plugin can be installed directly from the plugin repository or manually as follows 

1. Upload `product-tab-manager.zip` to the `/wp-content/plugins/` directory
2. Extract the zip file
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Once activated, you can manage default tabs from the admin menu Tab Manager

== Frequently Asked Questions ==

= Will this plugin work with variable product type? =

Yes, it works perfectly with all product types

= Can I create my own tabs ? =

Yes, you can create your own tabs in the premium version

= Can I completely hide all the three default woocommerce product tabs? =

Yes, you can hide all the three default product tabs or just the selected ones
